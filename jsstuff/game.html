Beach Clean Up!

  • The objective of the game is picking up trash and seashells that wash up on shore
  • Use the RED trash can to pick up the garbage (gray rectangle thing)
  • Use the BLUE trash can to pick up seashells to collect them!
  • If you use the wrong trashcan to pick up the wrong items a point is subtracted from your score
  • Use WASD to control the RED trash can
  • Use the ARROW KEYS to control the BLUE trash can
